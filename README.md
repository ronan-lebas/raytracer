# Raytracing Engine in Rust

## Presentation

I made this small project to learn about raytracing while refreshing my Rust knowledge. The basic code architecture was written by an LLM, with the important parts for actual raytracing logic left out, thus this project is more a LLM-generated lab / tutorial.

## Installation

Clone the code: `git clone git@github.com:ronan-lebas/raytracer.git`

Get the Rust toolchain if not already installed.

Then `cargo run` builds the code and generates the image (`output.ppm`). You can view the image with GIMP for instance.
